brew 安装 PHP 环境

安装brew

brew中文地址
brew常用命令

  • brew 安装\卸载软件
    brew install\uninstall XXX
  • brew 查看已安装的软件列表
    brew list
  • brew 查看软件的安装信息
    brew info XXX
  • brew 查看service列表状态
    brew services list

brew安装nginx

brew search nginx
brew install nginx

  • 启动nginx
    nginx
    启动后访问nginx http://localhost:8080
  • 关闭nginx服务
    nginx -s stop
  • nginx重启
    nginx -s reload
  • 查看nginx版本
    nginx -v
  • nginx.conf修改
  • 多站点配置
    新建sites目录，*.conf都在这里

brew安装php

brew search php
brew install php@7.3

  • 安装PHP扩展
    使用pecl安装
    • pecl安装教程
      tips: 此处若安装好命令不能用可能需要设置环境变量
    • 编译安装
      下载源码包；phpize；.configure；make && make install；在php.ini文件中添加扩展配置，重启php-fpm
  • 切换PHP版本

brew安装mysql

brew search php
brew install php@7.3

  • 安装完毕后设置数据库密码
    mysql_secure_installation
  • 开启msyql服务
    mysql.server start
  • 重启mysql服务
    mysql.server restart
  • 关闭mysql服务
    mysql.server stop
