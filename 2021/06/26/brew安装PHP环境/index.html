安装 brew

brew 中文地址
brew 常用命令

  • brew 安装\卸载软件
    brew install\uninstall XXX
  • brew 查看已安装的软件列表
    brew list
  • brew 查看软件的安装信息
    brew info XXX
  • brew 查看 service 列表状态
    brew services list

brew 安装 nginx

brew search nginx
brew install nginx

  • 启动 nginx
    nginx
    启动后访问 nginx http://localhost:8080
  • 关闭 nginx 服务
    nginx -s stop
  • nginx 重启
    nginx -s reload
  • 查看 nginx 版本
    nginx -v
  • nginx.conf 修改
  • 多站点配置
    新建 sites 目录，*.conf 都在这里

brew 安装 php

brew search php
brew install php@7.3

  • 安装 PHP 扩展
    使用 pecl 安装 - pecl 安装教程
    tips: 此处若安装好命令不能用可能需要设置环境变量 - 编译安装
    下载源码包；phpize；.configure；make && make install；在 php.ini 文件中添加扩展配置，重启 php-fpm
  • 切换 PHP 版本

brew 安装 mysql

brew search php
brew install php@7.3

  • 安装完毕后设置数据库密码
    mysql_secure_installation
  • 开启 msyql 服务
    mysql.server start
  • 重启 mysql 服务
    mysql.server restart
  • 关闭 mysql 服务
    mysql.server stop
