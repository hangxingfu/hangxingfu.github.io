    glide中文文档

    glide.yaml 完整配置示例

    • 修改yaml文件添加包， glide up

    dep

    Mac安装go依赖管理工具 dep

    go版本要>=1.9
    dep使用需要fanqiang

    安装完成,dep命令输出

    使用dep命令安装gin

    在 GOPATH/src 目录中使用 dep 命令

    新建任意目录，并在里面 dep init，成功后会生成两个文件（Gopkg.lock Gopkg.toml）和一个目录（vendor）

    • vendor 依赖包会下载到这里
    • Gopkg.lock 暂时不用管
    • Gopkg.toml 需要的依赖配置写在这里

    在 toml 文件中增加依赖配置

    之后执行 dep ensure 命令，会在vendor下安装包

  • go-mod引入包

    2021-07-25

    此方式需要将你的GO111MODULE设置成on,这样你就可以在任意目录下面开启你的项目了

    此时目录下就出现 go.mod 文件，打开查看

    新建文件main.go, 在import中写入想要引用的包
    我这里使用decimal包
    decimal包地址：https://pkg.go.dev/github.com/shopspring/decimal

    此时运行go run main.go会报错

    提示使用 go get github.com/shopspring/decimal 命令，使用会显示下载完成
    再运行就成功

  • brew 安装 PHP 环境

    2021-06-26 /
    • { favorite }

    安装brew

    brew中文地址
    brew常用命令

    • brew 安装\卸载软件
      brew install\uninstall XXX
    • brew 查看已安装的软件列表
      brew list
    • brew 查看软件的安装信息
      brew info XXX
    • brew 查看service列表状态
      brew services list

    brew安装nginx

    brew search nginx
    brew install nginx

    • 启动nginx
      nginx
      启动后访问nginx http://localhost:8080
    • 关闭nginx服务
      nginx -s stop
    • nginx重启
      nginx -s reload
    • 查看nginx版本
      nginx -v
    • nginx.conf修改
    • 多站点配置
      新建sites目录，*.conf都在这里

    brew安装php

    brew search php
    brew install php@7.3

    • 安装PHP扩展
      使用pecl安装
      • pecl安装教程
        tips: 此处若安装好命令不能用可能需要设置环境变量
      • 编译安装
        下载源码包；phpize；.configure；make && make install；在php.ini文件中添加扩展配置，重启php-fpm
    • 切换PHP版本

    brew安装mysql

    brew search php
    brew install php@7.3

    • 安装完毕后设置数据库密码
      mysql_secure_installation
    • 开启msyql服务
      mysql.server start
    • 重启mysql服务
      mysql.server restart
    • 关闭mysql服务
      mysql.server stop
